How to run a test

Running a test in the RubyMine is very simple. When the file with the test definition is open, just press the combination Ctrl + Shift + F10.

As section Environment mentions, if there are any errors in the test definition, instead of running the test the environment will present a list of errors in the view "Output". The errors have to be fixed for the test to be run.
